# Galcode Assets

This directory is the local asset drop zone for generated WebGAL projects.

The public repository does not redistribute copyrighted Live2D SDK/runtime files or official Live2D model assets. In particular, these files are intentionally not included:

- `vendor/webgal-mygo/packages/webgal/public/lib/live2d.min.js`
- `vendor/webgal-mygo/packages/webgal/public/lib/live2dcubismcore.min.js`
- `figure/mygo/`
- `figure/_mtn_exp/`
- generated WebGAL Live2D model folders under `vendor/webgal-mygo/packages/webgal/public/game/figure/live2d/`

If you need the complete demo assets, watch the project introduction video on the author's Bilibili account and follow the group download instructions there. Use those files only within the rights holder's permitted scope.
